# SMS Spam Detection

## Problem

Detect spam messages in SMS text using NLP and machine learning.

## Dataset

- 5,572 SMS messages
- 13% spam, 87% legitimate

## Results

- **Random Forest: 98.83% accuracy** (Winner!)
- Logistic Regression: 97.49%
- Naive Bayes: 96.95%

## Tech Stack

Python, Pandas, Scikit-learn, TF-IDF, Random Forest
